Thanks for considering contributing to this project! We welcome improvements, bug fixes, and new features via pull requests.   
## How to Submit Code

1. **Fork** the repository.
2. **Clone** your fork:  
   `git clone https://github.com/your-username/repo-name.git`
3. **Create a branch**:  
   `git switch -c feature/your-feature` or `git switch -c fix/issue-description`
4. Make your changes and **commit** them. 5. **Push** to your fork.
6. Open a **Pull Request (PR)** to the `main` branch of the original repo.
7. - Keep PRs focused on a single change.
- Include a clear description of what your PR does.
- Make sure your code follows the project's style.
- We'll review your PR and may ask for changes.
